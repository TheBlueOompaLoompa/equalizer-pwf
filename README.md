# Equalizer PWF
Inspired by my audiophile boyfriend who can't accept just decent quality. He also claims this is the one thing he needs to switch to Linux, so I'm obligated to make this. Equalizer PWF is a simple parametric EQ system for Linux/PipeWire. This is intended to be highly compatible with Equalizer APO config files and provide a similar experience for my platform of choice.

## Current Project Status
Baseline features are implemented, and the project is technically usable. Many filters are missing, and there is no graphic eq. There are only basic parametric functions.

Device selection is still not possible, it just autoconnects to whatever device your audio system connects an audio source to.
