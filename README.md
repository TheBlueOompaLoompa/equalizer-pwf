# Equalizer PWF
Inspired by my audiophile boyfriend who can't accept just decent quality. He also claims this is the one thing he needs to switch to Linux, so I'm obliged to make this. Equalizer PWF is a simple parametric EQ system for Linux/PipeWire. This is intended to be highly compatible with Equalizer APO config files and provide a similar experience for my platform of choice.

## Current Project Status
This isn't even out of fun prototyping phase. This is not usable as of now, please don't even hope this will work.
